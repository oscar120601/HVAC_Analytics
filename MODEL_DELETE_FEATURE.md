# 刪除已訓練模型功能

## 功能概述

在 UI 介面的兩個位置新增了「刪除已訓練模型」功能，方便管理模型檔案。

---

## 位置一：側邊欄「模型設定」區塊

### 適用時機
當你選擇「⚡ 最佳化模擬」模式時，側邊欄會顯示此功能。

### 使用方式
```
側邊欄
├─ ⚙️ 設定
│  ├─ 處理模式: ⚡ 最佳化模擬
│  ├─ 模型設定
│  │  ├─ [選擇已訓練模型 ▼]
│  │  │   model_20260203_092628.joblib
│  │  │
│  │  ├─ ────────────────
│  │  └─ [🗑️ 刪除選擇的模型]  ← 點擊這裡
│  │
│  └─ HVAC Analytics | Spec-Kit
```

### 操作流程
1. 選擇「⚡ 最佳化模擬」模式
2. 在「模型設定」區塊選擇要刪除的模型
3. 點擊「🗑️ 刪除選擇的模型」按鈕
4. 確認刪除後，系統會顯示「✅ 已刪除: model_name」
5. 重新整理頁面即可看到更新後的模型列表

---

## 位置二：主畫面「🔧 模型訓練」標籤頁

### 適用時機
在「⚡ 最佳化模擬」模式下，點擊「🔧 模型訓練」標籤頁。

### 使用方式

### 操作流程
1. 切換到「⚡ 最佳化模擬」模式
2. 選擇一個模型載入（或保持未選擇）
3. 點擊「🔧 模型訓練」標籤頁
4. 在「🗂️ 模型管理」區塊查看所有模型
5. 從下拉選單選擇要刪除的模型
6. 點擊「🗑️ 刪除模型」按鈕
7. 頁面會自動重新整理，顯示更新後的模型列表

---

## 功能特點

### 📋 模型列表資訊
- **模型名稱**: 完整檔案名稱
- **大小**: 以 MB 為單位顯示
- **建立時間**: 顯示年月日時分

### 🔄 即時更新
- 刪除後頁面會自動重新整理
- 側邊欄和主畫面的模型列表會同步更新

### ⚠️ 安全提醒
- 刪除前請確認模型已不再需要
- 刪除後無法復原，請謹慎操作
- 建議先下載備份重要模型

---

## 快速檢查清單

刪除模型前確認：
- [ ] 已備份重要模型（如需要）
- [ ] 確認模型不再使用
- [ ] 確認沒有正在使用該模型的優化任務

---

## 常見問題

### Q: 刪除模型後需要重新整理頁面嗎？
A: **側邊欄刪除**：需要手動重新整理頁面
**主畫面刪除**：會自動重新整理頁面

### Q: 可以一次刪除多個模型嗎？
A: 目前需要逐個刪除。如需批量刪除，可以直接到 `models/` 資料夾手動操作。

### Q: 刪除後的模型可以復原嗎？
A: **不可以**，刪除是永久性的。請務必確認後再操作。

### Q: 正在使用的模型可以刪除嗎？
A: **可以**，但不建議。如果刪除當前載入的模型，優化功能會無法使用，直到選擇新模型。
